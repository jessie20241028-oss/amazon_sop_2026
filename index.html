衰退期三步收缩公式
第1步：销量下滑信号出现
IF 周销量连续下降 > 15% THEN → 暂停竞品定向组 + 自动广告组 → 预算整体削减 50% → 仅保留 ACOS < 25% 的精准词组
第2步：连续5天日销 < 2件
THEN → 暂停所有手动词组，仅保留1个自动广告（$3/天） → 启动 Coupon 5-8% 配合广告做最后冲量
第3步：连续7天日销 < 1件
THEN → 全面停止广告投入 → 价格每周下调 5-8% 直至清空 → 归档数据，作为下季选品参考

好广告 vs 坏广告定义
✓ 好广告（可持续投入）
  • 每一次点击都在目标受众内
  • ACOS 持续低于品类盈亏平衡线
  • 自然排名因广告助推持续上升
  • 花费可被当周销售额覆盖并盈利
✗ 坏广告（48小时内处理）
  • 靠宽泛词买流量，买家不匹配
  • ACOS 长期超标却未触发止损
  • 花费高但自然排名无任何改善
  • 靠持续砸钱维持表面销量数字

四象限评估模型（CTR × CVR）
Q2 — 低CTR · 高CVR
页面问题广告
词义匹配（点了会买），但主图/价格让买家在搜索页就放弃了。属于"待拯救"，潜力大，不得直接否定。
→ 优化主图 / 降价对比 / 标题前40字符含核心词
Q1 — 高CTR · 高CVR
明星广告
CTR ≥ 0.3% 且 CVR ≥ 3% 且 ACOS < 30%。词义精准，Listing完整，买家点了即买。
→ 提预算30-50% + 提竞价15-20% + 移入核心精确组
Q4 — 低CTR · 低CVR
完全失效广告
无效花费的主要来源。曝光买了，买家既不点也不买。72小时内必须处理，无任何理由可拖延。
→ 72h内暂停或否定，预算100%转移至Q1
Q3 — 高CTR · 低CVR
词义偏差广告
买家被吸引点入，但进入Listing后不买。词义偏移或详情页说服力不足。ACOS通常虚高。
→ 拉搜索词报告 → 否定偏移词 → 检查A+/尺码表

IF 曝光(7天) > 5,000 （US: >4,000） AND CTR < 0.2% （US: <0.25%） AND 已持续 ≥ 7天 THEN 按优先级执行： → 步骤1：对比同位置竞品主图，若明显弱势 → 提交换图任务（7天内） → 步骤2：价格比同页面前5名高 >15% → 降价5%观察3天 （US: 降价8%） → 步骤3：拉搜索词报告，CTR < 0.1% 的词 → 精确否定 → 步骤4：14天内 CTR 仍 < 0.2% → 暂停该广告组 禁止动作：不得通过提高竞价强行刷曝光位置
02低曝光 · 高转化 · 立即处理
▼
IF 曝光(7天) < 1,000 AND CVR > 5% （US: >4%） AND 有 ≥ 2 笔转化 THEN → 步骤1：竞价低于建议50%+ → 提至建议竞价×1.0 → 步骤2：14:00前预算耗尽 → 预算提升40% → 步骤3：词组匹配升级为精确匹配，独立广告组保护 → 步骤4：挖掘10-20个近义长尾词扩大覆盖 → 步骤5：竞价策略切换"动态-升降" + 首页溢价 +30% （US: +40%） 禁止动作：不得因曝光低就否定此词，这是最有价值的词
03高点击 · 低转化 · 48小时内处理
▼
IF 点击数(7天) > 30 （US: >25） AND CVR < 1.5% AND 花费 > $15 （US: >$20） THEN 判断根因类型： 【类型A：词义偏移】 → 拉搜索词报告，查Top10词实际意图 → 词义不符 → 精确否定，观察7天 【类型B：Listing说服力不足】 → 检查：Review<5 / 无A+ / 缺尺码表 / 差评含"图不符实" → 修复后标注"待验证"，14天后复查CVR 【类型C：价格锚点失效】 → 加购比例 < 5% 且 CTR > 0.4% → 降价8-10%或设5%优惠券 终止：21天CVR持续<1.5% → 暂停该广告组
04高花费 · 零转化 · 24小时内 · 最高危
▼
⚑
花费 > $30 且 0 转化 → 不经讨论，当天暂停，无任何理由延迟。
IF 花费(7天) > $12 （US: >$18） AND 转化 = 0 THEN 分支判断： 【分支A：点击>10 但CVR=0 → Listing根本性问题】 → 立即暂停广告（停止流量进入烂页面） → 48小时内修复Listing最严重缺陷 → 修复后用$5/天小预算验证7天 【分支B：点击≤5 → 词义完全不匹配】 → 24小时内否定所有触发曝光的搜索词 → 自动广告设置"紧密匹配"，关闭"关联商品" 【分支C：点击5-10 → 词义轻度偏移】 → 竞价降低40%，预算降至$3/天 → 7天仍无转化 → 执行分支B止损
05ACOS持续超标 · 不收敛 · 72小时内处理
▼
IF ACOS(7天) > 45% （US: >40%） AND 连续 ≥ 2 个完整周期无改善 THEN 三刀止血法： 【第一刀：词级手术】 → 花费前20% 且 转化后50% 的词 → 降价30%或暂停 → 转化前20%的词 → 竞价提升10% 【第二刀：匹配方式收紧】 → 宽泛→词组 / ACOS>50%的词组→精确 → 自动广告仅保留"紧密匹配" 【第三刀：预算重分配】 → 该Campaign预算削减40% → 释放预算→转至ACOS<25%的Campaign 复查：7天后若ACOS无改善 → 暂停整个Campaign，进入"问题品待处置"清单
06曝光点击俱低（广告隐形） · 3天内处理
▼
IF 曝光(7天) < 500 AND 点击(7天) < 5 AND 广告已运行 > 7天 THEN 按序排查： → 排查1：竞价 < 建议×60% → 提至建议×90%，观察3天 → 排查2：词月搜索量 < 300 → 该词需求不存在，直接暂停换词 → 排查3：ASIN有Policy Warning/差评率>15% → 先处理合规问题 → 排查4：同关键词首页全为FBA+Review>500 → 转长尾词或竞品定向
07转化波动剧烈 · 7天内追踪
▼
IF 相邻7天周期转化数差异 > 50% AND 无节日/促销原因 AND 连续 ≥ 3 个周期 THEN → 检查1：FBM库存 < 3件 → 补货，广告自动回稳 → 检查2：过去21天价格变动 > 3次 → 锁价14天建立基线 → 检查3：竞品价格比你低 > 20% → 跟价或转攻长尾词 → 检查4：同组内跨品类词（如"连衣裙"+"礼服"）→ 拆分广告组
08自动广告失控 · 词义偏移 · 48小时内处理
▼
IF 自动广告组 7天花费 > $20 AND 不相关词曝光占比 > 40% AND ACOS > 50% THEN 自动广告清洗四步法： → 步骤1：立即竞价降低50%，日预算降至$5 → 步骤2：花费Top20词逐一判断： 有转化相关词 → 移至手动精确组 + 自动组否定 无转化相关词 → 自动组否定 不相关词 → 广告组级+Campaign级双重否定 → 步骤3：关闭"宽泛匹配"和"关联商品"，仅保留"紧密匹配" → 步骤4：自动广告预算占比上限 ≤ 总预算20% Xnurta规则：单词花费>$8且转化=0 → 自动否定 单词ACOS<20%且转化≥2 → 自动移入手动精确组
09节日后ACOS骤升 · 24小时内处理
▼
IF 节日结束后第3-7天 AND ACOS 比节日前基准线高 > 20个百分点 AND 竞价未做调整 THEN → 节日后24h内：所有提价词恢复至节日前水平 → 节日专属词（如"Halloween dress"）→ 暂停或降价80% → 节日后3天：否定所有节日特定词（无长期搜索价值） → 节日后7天：自然排名若稳定上升 → 词有长期价值，保留 若跌回原位 → 归入"季节词库"明年复用 提前防范：Xnurta规则：节日+3天后ACOS>均值×1.3 → 自动降价20%
10竞品词ACOS极高 · 无转化 · 7天内处理
▼
IF 竞品ASIN定向组 ACOS > 60% （US: >55%） AND 7天花费 > $14，转化 < 1 THEN → 判断1：强势竞品（Review>200 + BSR前100）→ 放弃，转长尾词 → 判断2：弱势竞品（Review<50）但价格高于对方 → 先跟价再测7天 → 判断3：品牌词 → 停止，品牌词竞价高且转化低 铁律：竞品词预算不超总预算10% / 单ASIN 7天无转化→立即暂停
11单词霸占全部预算 · 立即处理
▼
IF 单一词/组 消耗 > Campaign总预算60% AND 其余词因预算耗尽频繁断投 THEN → 步骤1：高消耗词独立为单独Campaign，设专属日预算 → 步骤2：其余词恢复独立预算（独立Campaign或Ad Group） → 步骤3：评估霸占词价值： ACOS<25% → 独立Campaign全力投入 ACOS 25-40% → 设竞价上限防无上限消耗 ACOS>40% → 降竞价40%，强制控制消耗比例
12长尾词堆积 · 词库臃肿 · 月度清洗
▼
IF 单组词数 > 50 AND 60天内0点击词 > 30% AND 整体ACOS > 35% THEN ABC三级分类清洗（60天数据）： A级保留：转化≥2 且 ACOS<35% → 保留，考虑移入精确组 B级观察：有点击0转化 且 花费<$3 → 竞价降50%，再观察30天 C级清除：60天0点击 → 暂停（不删除，保留数据） 60天花费>$5且0转化 → 精确否定 → 清洗后上限：精确匹配≤20个/组，词组匹配≤30个/组 → 每月第一个周一例行执行
